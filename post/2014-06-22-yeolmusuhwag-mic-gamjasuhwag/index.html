<그림 3: 감자>
어제 비가 오긴 했지만, 옆밭 할머니께 여쭤보니 비닐 멀칭을 했으면,
땅이 많이 안 젖어서 괜찮다고 하셔서 나머지 두 이랑도 오늘 수확했어요.
그리고, 다음주에 서리태 심기 위해서 밭도 정리하고, 밑거름도 주고..

<표 1: 수확량 비교표>

고찰

표본 집단이 적어서 오류가 있을 수는 있지만, 생산량을 기준으로 봤을 때 "비닐 + 순자르기 O", "신문지 + 순자르기 X" 가 좋음. (비닐 쪽이 무조건 좋을 것으로 생각되었는데...)
만약 오류가 없다면, 순자르기를 안하는 "신문지 + 순자르기 X"가 생산성이 더 좋아보임. 내년엔 "비닐 + 순자르기 O"와 "신문지 + 순자르기 X" 두 방법으로 비교하고 최종적으로 생산방법을 선택해야할 것 같음.
