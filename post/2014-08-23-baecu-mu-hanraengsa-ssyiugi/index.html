2014-08-23 배추, 무 한랭사 씌우기

Tweet

한 일

  • 배추, 무 한랭사 씌우기
  • 배추, 무 재파종 (싹 안난 곳)
  • 일부 서리태 2차 순 자르기 (1m 이하로)

다음 할 일

  • 고구마 순 자르기


<그림 1: 텃밭전경>




<그림 2: 서리태>
서리태가 너무 크면 도복된다고 하는데 일부 서리태가 도복이 되려고 해서 2차 순 자르기 함.
또, 순 자르기를 하면 양분이 꽃으로 더 많이 가고 결국 빨리 꼬투리가 생긴다고 함.


<그림 3: 고구마>



<그림 4: 배추>
지난주 파종한 배추는 3/28 밖에 발아가 안되어 재파종하고, 한랭사 씌움.
지금(9/13) 생각해보면, 발아가 안된게 아니라 벌레가 배추 싹을 다 먹어치운 것 같음. ㅡㅡ;;
결국, 싫지만 토양살충제를 밭 만들때 뿌렸어야 했던 것 같다. ㅡㅡ;;





<그림 5: 무>
무는 29/39 로 거의 발아가 됨. 발아 안된 곳은 재파종하고, 한랭사 씌움.
