<그림 2: 서리태>
감자밭에 서리태를 심어야 해서 다른 밭보단 2주정도 늦긴 했지만(작년에도 6월말에 심었지만..),
지난주 수확한 감자밭을 정리하고, 오늘 서리태를 심었어요.
이번에도 감자처럼(올해 감자에서 재미를 봐서.. ^^;;) 두 이랑은 비닐, 한 이랑은 신문지 멀칭을 하고,
어제 하루 침종(물에 불린)한 서리태를 사진과 같이 20cm 간격을 두고 2개씩 심었습니다.
다음주에 새들한테 안 먹히고 싹이 잘 나와야할텐데.. 걱정반 기대반입니다. ^^
% 추가로 원래는 세이랑만 심으려고 했는데 콩이 너무 많이 남아서 멀칭안한 밭도 하나 만들었습니다.


<그림 3: 옥수수>
옥수수는 지난주에 웃거름을 안 줘서 오늘 웃거름(지난번에 만든 난각비료)를 줬어요.
수염이 지난주보다 더 많이 나온게 7/19일정도 수확하려고 했는데 더 빨라질 수도 있을 것 같아요. ^^



<그림 4: 당근>
당근은 사진처럼 5개정도 뽑아보니까 큰 건 20cm 이고, 보통 10 ~ 15cm 정도 되는게 다음주에 수확하면 될 것 같아요.
여담으로 장보면서 마트 시세를 보니 980원/개 * 개/350g = 2,800원/kg 이더라구요. 당근이 감자보다 좀 더 받는.. ^^;;
